### 一、 實證模型設計 (Empirical Model Design)

本研究利用多元迴歸（Multiple Regression）與遞延期模型（Lagged Model），針對您的三個假說進行驗證。為求精確衡量企業投資於水電資源的實質衝擊，本模型以**「水電設備資本支出 ($Water\_Energy\_CAPEX$)」**取代容易產生雜訊的整體現金流量表資本支出。

#### 模型 1：基礎層次－直接降本增值效應
*   **驗證目的**：檢驗單一資源管理（水或電）的表現，是否能直接降低營運成本並提升公司價值。
*   **實證模型**：
    $$Y_{i,t} = \alpha + \beta_1 Water_{i,t} + \beta_2 Energy_{i,t} + \gamma Controls_{i,t} + \varepsilon_{i,t}$$
    *(預期結果：若 $Y$ 為「營業費用率」，預期 $\beta_1, \beta_2 < 0$；若 $Y$ 為「Tobin's Q」，預期 $\beta_1, \beta_2 > 0$)*

#### 模型 2：延伸層次一－水能互補綜效 (Interaction Effect)
*   **驗證目的**：檢驗企業「同時」高度投入再生水與再生能源，是否會產生 1+1>2 的外溢綜效。
*   **實證模型**：
    $$Y_{i,t} = \alpha + \beta_1 Water_{i,t} + \beta_2 Energy_{i,t} + \mathbf{\beta_3 (Water_{i,t} \times Energy_{i,t})} + \gamma Controls_{i,t} + \varepsilon_{i,t}$$
    *(預期結果：關鍵在於交乘項係數 $\beta_3$。若 $Y$ 為「Tobin's Q」且 $\beta_3$ 顯著為正，代表雙軸管理能創造顯著的加乘溢酬)*

#### 模型 3：延伸層次二－專項資本支出的時間落差效應 (Time-Lag / U-Shape Effect)
*   **驗證目的**：檢驗建置「水電相關設備」的專項資本支出，是否在短期造成帳面成本上升/價值下降，但在中長期（例如完工啟用後）迎來降本增值的「U型轉折點」。
*   **實證模型（遞延期測試）**：將應變數向後遞延 1 期、3 期、5 期（$k = 1, 3, 5$）。
    $$Y_{i, t+k} = \alpha + \mathbf{\beta_1 Water\_Energy\_CAPEX_{i,t}} + \beta_2 Water_{i,t} + \beta_3 Energy_{i,t} + \gamma Controls_{i,t} + \varepsilon_{i,t}$$
*   **預期結果（U 型翻轉檢定）**：
    *   **當 $Y$ 為「公司價值 (Tobins Q)」時**：預期 $t+1$ 期的 $\beta_1$ 顯著為**負**（短期資本支出侵蝕估值）；但在 $t+3$ 或 $t+5$ 期的 $\beta_1$ 翻轉為顯著為**正**（長期綠電/再生水發酵）。
    *   **當 $Y$ 為「營業費用率」時**：預期 $t+1$ 期的 $\beta_1$ 顯著為**正**（短期折舊與維護成本增加）；但在 $t+3$ 或 $t+5$ 期的 $\beta_1$ 翻轉為顯著為**負**（長期資源循環帶來經常性開銷下降）。

### 二、 TEJ 資料庫撈取實務與變數對應總結

以下是您在實務上操作 TEJ 資料庫時，對應上述模型的確切資料庫路徑與欄位名稱：

#### 1. 自變數 (Independent Variables, X)
*   **水資源與再生水管理 ($Water_t$)**
    *   **TEJ 路徑**：`TESG 永續發展解決方案` $\rightarrow$ `TESG永續資料集` $\rightarrow$ `環境構面` $\rightarrow$ **「企業用水量」資料庫**。
    *   **撈取欄位**：**`水回收率%`** 或 **`製程水回收率%`**。
*   **再生能源管理 ($Energy_t$)**
    *   **TEJ 路徑**：`TESG 永續發展解決方案` $\rightarrow$ `TESG永續資料集` $\rightarrow$ `環境構面` $\rightarrow$ **「企業使用綠電情形」資料庫**。
    *   **撈取欄位**：**`再生能源使用率`**。
*   **水電設備專項資本支出 ($Water\_Energy\_CAPEX_t$)** *(用於模型 3)*
    *   **TEJ 路徑**：`TEJ IFRS Finance-國際會計準則` $\rightarrow$ `以合併為主財務` $\rightarrow$ **資產負債表明細**。
    *   **撈取欄位**：**`水電設備成本`**。
    *   **計算方式**：撈取後，以「當期水電設備成本 - 前期水電設備成本」求得當年度新增的水電專項資本支出。
    *   *(替代方案庫)*：若需觀察極大額的特定建置案，可至 **「固定資產異動」資料庫**，利用「標的物」欄位搜尋（如：太陽能、廢水、再生水等），加總其「總價款」。

#### 2. 應變數 (Dependent Variables, Y)
*   **降低營運成本 ($Cost_{t, t+k}$)**
    *   **TEJ 路徑**：`TEJ IFRS Finance-國際會計準則` $\rightarrow$ **「財務比率」資料庫**。
    *   **撈取欄位**：**`營業費用率`** 或 **`營業利益率`**。
*   **增加公司價值 ($Value_{t, t+k}$)**
    *   **TEJ 路徑**：`TEJ IFRS Finance-國際會計準則` $\rightarrow$ **「財務比率」資料庫**。
    *   **撈取欄位**：**`Tobins Q`** 或 **`Tobins Q(A)`**。

#### 3. 控制變數 (Control Variables, CV)
*   **技術研發投入**：**`研究發展費(年報)`** 或財報中的 **`研究發展費用率`**（用於控制企業因研發技術升級對估值的干擾）。
*   **企業規模**：財報中的 **`資產總額`**（取自然對數 LN，控制大公司資源優勢偏誤）。
*   **財務風險/結構**：財報中的 **`負債比率`**（控制高槓桿帶來的財務與資金風險干擾）。

### 三、 資料整併與實務撈取提醒 (Tips)

1.  **資料對齊基準**：從 TEJ 下載 Excel/CSV 檔案時，請務必將**「公司代碼」**與**「年月 / 年度」**兩個欄位勾選匯出。不同資料庫（如 TESG、IFRS 財務報表）的資料，必須依賴這兩個主鍵（Key）才能在 STATA 或 Python 裡面正確合併 (Merge)。
2.  **樣本期間**：由於 TEJ 中「企業用水量」資料收錄自 **2013 年起**，「企業使用綠電情形」資料收錄自 **2021 年起**，建議您的樣本期間可以鎖定在綠電資料較完整的近年份，或將早期未揭露綠電的年份數值視為 0（假設當年尚未導入），以此構成不平衡面板資料（Unbalanced Panel）。
3.  **極端值處理 (Winsorization)**：針對財報資料中的 Tobin's Q 與資本支出新增額，建議實證跑模型前進行前後 1% 的縮尾處理（Winsorize），以排除資料中少數極端值對統計顯著性的扭曲。
